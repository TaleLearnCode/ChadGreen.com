## About VSLive! Las Vegas

Visual Studio Live! is a long-running developer conference series dedicated to practical training and education for Microsoft developers. The Las Vegas edition is one of the flagship events in the series, covering topics from cloud architecture and Azure to .NET development, DevOps, and modern software engineering.
