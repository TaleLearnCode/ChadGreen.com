## Louisville .NET Meetup

### About

The Louisville .NET Meetup is a community-driven monthly gathering for .NET developers in the Louisville area. The meetup focuses on sharing knowledge about modern .NET development, cloud technologies, software architecture, and emerging trends in the .NET ecosystem.
