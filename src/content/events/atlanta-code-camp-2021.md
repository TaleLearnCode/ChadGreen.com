## About Atlanta Code Camp

Atlanta Code Camp is a free, community-organized developer conference held annually in the Atlanta area. The event brings together developers and technology professionals for a full day of sessions covering a wide range of software development topics, from cloud architecture and DevOps to software craftsmanship and emerging technologies.
