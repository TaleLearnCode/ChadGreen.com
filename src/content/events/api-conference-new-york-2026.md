API Conference New York is a premier gathering for modern API strategy, connectivity, and AI‑driven governance, hosted at the New York Marriott at the Brooklyn Bridge from September 28 – October 2, 2026. Designed for architects, developers, platform engineers, and technology leaders, the event focuses on how APIs now serve as the governed access layer to enterprise capabilities—powering applications, automation, and increasingly, AI agents.

The conference explores how organizations can design, secure, and operate APIs that support distributed systems, intelligent automation, and large‑scale enterprise connectivity. Core themes include API Governance & Operations, APIs for AI & Agentic Systems, API Design & Development, API Platform Engineering, and Identity, Access & API Security.

Attendees can participate in two main conference days (Sept 29–30), hands‑on workshop days (Sept 28 & Oct 1), and optional bootcamps. With 10+ years of API events, five specialized tracks, and a global community of 1,200+ participants, the conference blends deep technical content with practical, real‑world patterns for building secure, scalable, and human‑ and agent‑friendly APIs.

API Conference New York is part of a multi‑conference week, allowing participants to move seamlessly between sister events and gain broader perspectives across cloud, DevOps, and platform engineering. Whether attending onsite in Brooklyn or joining remotely, participants benefit from expert‑led sessions, interactive experiences, and six months of access to session recordings.
