Visual Studio Live! (VSLive!) Las Vegas 2026 brings together .NET and Microsoft ecosystem professionals for five immersive days of cutting-edge training. The conference features over 60 sessions delivered by industry-renowned experts and Microsoft insiders, covering the latest advancements in Visual Studio 2026, .NET 9, ASP.NET Core, Generative AI, GitHub Copilot, .NET MAUI, Azure, Blazor, C#, and modern data solutions.

## Conference Highlights

Led by industry experts and Microsoft Program Managers, sessions go beyond theory to deliver real-world insights and practical guidance across native client development, web, cloud, AI, and data technologies. The conference includes interactive live demos, deep-dive workshops, and hands-on labs designed to transform the way you build software.

### Track Topics

- **Modern Software Engineering:** DevOps, GitHub, testing, and agile practices
- **Cloud, Containers & Microservices:** Azure, serverless, container orchestration
- **Cutting-Edge AI:** Generative AI, machine learning, Azure AI services
- **Data & Analytics:** SQL Server, Azure SQL, Entity Framework, data modernization
- **Developing New Experiences:** MAUI, Blazor, web development
- **Human Factors in Engineering:** Team dynamics, communication, career development
- **The Core of .NET:** C# 13, .NET 9, performance optimization
- **Full Stack Web Development:** ASP.NET Core, Blazor, modern web patterns

### Networking & Activities

Beyond the technical sessions, VSLive! Las Vegas offers exceptional networking opportunities, including:
- Conference welcome reception
- Lunch-and-learn sessions with Microsoft experts
- Evening networking events
- Birds-of-a-feather discussions
- Access to the Las Vegas Strip's entertainment and attractions
