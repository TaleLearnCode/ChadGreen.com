## Building Serverless Solutions with Azure and .NET

Immerse yourself in a comprehensive full-day workshop that delves into the transformative world of serverless computing using Azure and .NET. Throughout this intensive day, you will participate in practical activities illuminating the fundamental concepts, tools, and best practices for architecting serverless applications on the Azure platform using the .NET framework.

This workshop focuses on building a real-world business scenario for Cool Revive Technologies—a company breathing new life into old refrigerators. You will build a solution for Cool Revive's remanufacturing process through hands-on labs, experiencing first-hand how serverless technologies can modernize and automate complex business workflows.

### Learning Objectives

- Understand the key Azure serverless services and how they work together to form a cohesive, event-driven architecture.
- Apply serverless design patterns to architect a real-world business solution using Azure Functions, Event Grid, Service Bus, and Cosmos DB.
- Deploy and manage a serverless solution using CI/CD best practices, and apply security and monitoring strategies to keep it production-ready.
