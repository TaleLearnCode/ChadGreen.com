## Future API Documentation

API documentation has to serve both people and machine consumers. This session explores how to keep docs aligned with the implementation while making them easier to consume and maintain.

### What this session covers

- Documentation patterns that reduce drift.
- How AI changes the expectations for API docs.
- Ways to keep docs useful for both humans and automation.
