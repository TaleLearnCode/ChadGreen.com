## Improving IaC: Infrastructure as Documentation

Infrastructure-as-code can be more than a deployment tool. This session focuses on making IaC readable, durable, and useful as living documentation for cloud and platform work.

### What this session covers

- Treating infrastructure definitions as source-of-truth documentation.
- Using naming and structure to communicate intent.
- Keeping documentation and deployed systems in sync.
