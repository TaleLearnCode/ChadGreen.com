API documentation is no longer just for humans reading pages by hand. As AI assistants and agents become part of the development workflow, documentation needs to serve automated consumers without losing clarity for people.

This session looks at practical ways to document APIs so they remain accurate, discoverable, and maintainable as both human and machine consumers evolve.
